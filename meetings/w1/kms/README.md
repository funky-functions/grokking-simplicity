함수형 프로그래밍~!

1. 수학 함수를 사용하고 부수 효과를 피하는 것이 특징인 프로그래밍 패러다임
2. 부수 효과 없이 순수 함수만 사용하는 프로그래밍 스타일

소스 코드를 3가지로 구분
1. 액션   - 실행 시점이나 횟수 또는 둘 다에 의존
2. 계산   - 입력값으로 출력값을 만드는 것
3. 데이터 - 이벤트에 대해 기록한 사실. 같은 데이터를 여러 형태로 해석할 수 있음


※ 같이 생각해볼 문제

예시) 시간에 대한 형태 or 데이터의 타입
