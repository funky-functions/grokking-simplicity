### 중요하다고 생각하는 것

- 액션과 계산, 데이터의 구분하는 것이 중요하다
- 액션을 최대한 나누어서 부수효과를 줄인다
- 액션 나누는 것은 계산으로 바꾼다 

**[ 계산으로 바꿀 때 순서 ]**

1. 계산 코드를 찾아 빼기
2. 새 함수에 암묵적 입력과 출력 찾기
3. 압묵적 입력은 `인자` 암묵적 출력은 `리턴 값` 으로 바꾼다

인자와 리턴값은 바뀌지 않는 `불변 값` 이라는게 중요하다
